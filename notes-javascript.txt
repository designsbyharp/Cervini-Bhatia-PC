 Importing files into main javascript file:
 Example used will be helloWorld.js and Index.js


 Inside helloWorld.js:

 Must export function/class/code to be used in another file:
 export default helloWorld;

 then class/function/code desired to use into desired file:
 import helloWorld from './hello-world.js';

 then call: helloWorld();

 in index.html use the dist js file:
 <script src="./dist/bundle.js"></script>
